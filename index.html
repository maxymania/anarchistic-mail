Introduction

Anarchistic Mail (in short A-Mail) is a Project, that makes a set of Protocols for a seperate E-Mail-Like Mail System with Privacy in mind.

It is a bit similar to Dark Mail, but simpler, and seperate, not complement, to the E-Mail standard.

That means, that you can't send a A-Mail to an E-Mail-Address, nor an E-Mail to an A-Mail-Address (Though there will be Gateways for this)

One goal is to Provide:

  • Full encryption of both text and metadata
  • Remailer-like capabilities (Type III)
  • Anonymous sending
  • Deniability (if wanted)

Short Term Goals

  • Design the Protocols, Formats and Encryption shemes for A-Mail
  • Create Reference implementations
  • Create an A-Mail client

Long Term Goals

  • Create a Gateway (-software) to receive E-Mails with an A-Mail Account.
  • Create a Gateway (-software) to send A-Mails to E-Mail-Addresses.

Formats

  • Mail Relay Message Format

Schemes

  • ECFB
  • Seccon

Protocols

  • Service Information Protocol
  • Very Secure Encryption Protocol (VSEP)
  • PUSH Protocol
  • PULL Protocol
  • Key Offer Protocol
  • Key Poker Protocol
